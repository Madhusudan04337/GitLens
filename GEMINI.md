# GitHub Dev Card Generator

## Project Overview
An expert Python developer building a GitHub Dev Card Generator.

## Tech Stack
- **Agent Orchestration:** Google ADK
- **Tools:** MCP (FastMCP)
- **LLM:** Gemini 2.5 Flash
- **Backend:** FastAPI
- **Frontend:** React/HTML
- **Deployment:** Google Cloud Run
- **Dependency Management:** `uv`

## Workflow Mandates
- **Strict Adherence:** Only execute prompts provided directly by the user.
- **Auto-Commit/Push:** Every build/prompt completion must be committed and pushed to the repository.
- **Code Style:** Clean, modular Python.
